# 🎮 Connect Four

A simple and interactive implementation of the classic **Connect Four** game. Players take turns dropping their pieces into a 7×6 grid, aiming to connect four of their pieces horizontally, vertically, or diagonally before their opponent.

## ✨ Features

- Two-player gameplay
- Interactive game board
- Automatic win detection
- Draw detection
- Input validation
- Clean and easy-to-understand code

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone <https://github.com/Shoaib1152/Connect_four_dots/>
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation is required.

## 🎯 How to Play

- Player 1 and Player 2 take turns selecting a column.
- The piece automatically falls to the lowest available position.
- The first player to connect **four consecutive pieces** horizontally, vertically, or diagonally wins.
- If the board fills up without a winner, the game ends in a draw.
